무선서비스 표준안내서

※ 표준안내서 확인버튼을 눌러주셔야 정상적으로 접수가 완료됩니다.

2022년 09월 26일

가입자 : 홍길동 (서명/인)

가입내역

※ 이벤트 유의사항

  • 1. 아래 일부 요금제에 한해 180일 유지기간이 있으며 180일 안에 중도 해지시 [요금 할인 금액 x 사용 개월 수]만큼 위약금이 청구됩니다.
  • 2. 특별 할인 요금제의 경우 개통일자 기준으로 해당 월의 말일까지 개통 완료한 고객님 대상으로 할인 요금제가 적용됩니다.
    • 말일에 접수하여 익월에 개통 완료된 고객님(개통 지연 사유 포함)은 익월에 진행되는 이벤트가 또한 정상 요금제로 적용됩니다.
    • ex) 1월 10일 접수, 1월 14일 개통 → 1월 이벤트 적용
    • ex) 1월 31일 접수, 2월 1일 개통 → 2월 이벤트 적용
  • 3. 발송되는 표준안내서는 신청일자 기준 이벤트 할인가가 표기되며, 실제 청구 시에는 개통완료 일자로 적용되어 할인금액이 상이할 수 있습니다.
  • 4. 이벤트 및 사은품은 당사의 사정에 따라 사전고지 없이 조기 종료, 변경될 수 있습니다.
  • 5. 특별 할인 기간 만료 이후 기본요금 금액으로 매월 청구됩니다.

위약금

※ 위약금이 부과되는 경우

  • 1. 요금약정할인 받고 약정 기간 내 해지 시
  • 2. 요금약정할인 받고 약정 기간 내 요금제 변경
  • 3. 공시지원금을 받은 후 약정 기간 내 해지 시
  • 4. 공시지원금을 받은 후 요금제 하향 변경 시
※ 해지시 예약 위약금

공시지원금

12개월 약정시

= 없음

24개월 약정시

= 6개월 이내 해지 및 요금제 변경시 공시지원금 100% 부과 = 6개월 이후 : {공시지원금 × (잔여 기간 / 약정기간 - 180일)}

요금약정할인

6개월 약정 시

= 6개월 이하 : 할인금 x 할인 개월 수

12개월 약정 시

= 3개월 이하 : (약정금 / 100%) x 100%
= 4개월 이상 9개월 이하 : (약정금 / 100%) x 50%
= 10개월 이상 12개월 이하 : (약정금 / 100%) x 30%

24개월 약정일시

= 6개월 이하 : (약정금 / 100%) x 100%
= 7개월 이상 12개월 이하 : (약정금 / 100%) x 50%
= 13개월 이상 16개월 이하 : (약정금 / 100%) x 30%
= 17개월 이상 20개월 이하 : - (약정금 / 100%) x 30%
= 21개월 이상 24개월 이하 : - (약정금 / 100%) x 30%

※ 알려드립니다

  • - 이벤트를 통해 가입하신 특가할인 요금제는 다른 요금제로 변경 시 특가할인은 재적용되지 않습니다.
  • - 본 표준안내서는 이용요금,예상위약금,약정조건,요금할인 등의 중요한 사항을 정리한 것이며, 그 밖의 사항은 가입신청서 등을 반드시 확인하시기 바랍니다.
  • - 해외에서는 데이터 사용 요금이 과다 발생될 수 있습니다. 출국 전 고객센터로 문의하여 정액 로밍 요금제에 가입 또는 로밍 데이터 차단을 설정해야 합니다.
  • - 청구 첫 달은 사용일수에 따라 일할 계산이 됩니다.
  • - 데이터를 (음성,문자 포함) 많이 사용한 상태에서 요금제를 변경할 경우 일할 계산 방식으로 인해 데이터 초과사용에 따른 추가 요금이 발생할 수 있습니다.
